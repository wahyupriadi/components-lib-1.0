### variant

There are four button variants. Use `variant` prop to assign button variant.<br />
Default value is "primary".

<Preview>
  <Story id="atom-button--variant" />
</Preview>

### size

There are three button sizes. Use `size` prop to assign button size.<br />
Default value is "medium".

<Preview>
  <Story id="atom-button--size" />
</Preview>

### disabled

use `disabled` prop to disable the button.

  <Story id="atom-button--disabled" />

### spinner

`spinner` prop controls the appearance of the loading spinner. When it's **true**, the spinner will show, and vice versa.
