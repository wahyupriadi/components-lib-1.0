# Tabs
`Tabs` component navigate between views. Line under `Tabs` header indicates which view is currently active.

## Stories

### Long Tabs
Long Tabs variant makes tabs scrollable if tabs total width exceed container or device width.
`longTabs` prop also required to activate this variant.

  <Story id="molecules-tabs--long-tabs" />

## Component Props

### underlineSize
`underlineSize` prop set the thickness of line under tab.

<Preview>
  <Story id="molecules-tabs--underline-size" />
</Preview>

### underlineColor
`underlineColor` prop set the color of line under tab.
The value can be a color hex or [this color pallete](/?path=/docs/utils-color--list).

<Preview>
  <Story id="molecules-tabs--underline-color" />
</Preview>

### dots
`dots` prop replaces line under tabs with 3 dots.

<Preview>
  <Story id="molecules-tabs--dots" />
</Preview>

### responsive
`responsive` prop makes overall tabs width stretches to container or device width.

<Preview>
  <Story id="molecules-tabs--responsive" />
</Preview>

### tabsMargin
`tabsMargin` prop gives margin space between tabs.

<Preview>
  <Story id="molecules-tabs--tabs-margin" />
</Preview>

### tabsPadding
`tabsPadding` prop gives padding space between inside a tab.

<Preview>
  <Story id="molecules-tabs--tabs-padding" />
</Preview>

### noGutter
`noGutter` prop combines with `tabsPadding` prop will nullify left padding of first tab and right padding of last tab.
